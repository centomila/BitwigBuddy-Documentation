# File Actions

## Available Actions

| Action ID           | Action Name                |
| ------------------- | -------------------------- |
| New                 | New Project                |
| Open                | Open...                    |
| Save                | Save                       |
| Save as             | Save as...                 |
| Close               | Close                      |
| Quit                | Quit                       |
| new_from_template   | New From Template...       |
| save_as_template    | Save as Template...        |
| add_to_library      | Save to Library...         |
| import_wavetables   | Import Wavetables...       |
| import_impulses     | Import Impulses...         |
